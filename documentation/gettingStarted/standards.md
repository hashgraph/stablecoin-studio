# Standards & Compliance

## Hedera Standards (HIPs)

* **HIP-10**: Token Metadata standard.
* **HIP-206**: HTS & Smart Contract interoperability.
* **HIP-482**: Transparent Proxies on Hedera.

## OpenZeppelin Mapping

| Hedera Feature | OpenZeppelin Equivalent |
| :--- | :--- |
| HTS Native Token | [ERC-20](https://docs.openzeppelin.com/contracts/5.x/api/token/erc20) |
| Proxy Logic | [TransparentUpgradeableProxy](https://docs.openzeppelin.com/contracts/5.x/api/proxy) |
| Roles | [AccessControl](https://docs.openzeppelin.com/contracts/5.x/api/access) |
